# Sistema de archivado de notas con interfaz gráfica

Consiste en un sistema que mantiene registro de todas las notas, e intreactúa con el usuario mediante una interfaz gráfica

### Features
- Actualizar nota: sobreescribe la nota según el título. Si cambia el título, crea una nota nueva. Si no, sobreescribe la nueva con el contenido modificado
- Eliminar una nota seleccionada
- Eliminar todas las notas (con un aviso)
- Sistema de login y registro
- Búsqueda en tiempo real por título y contenido
- Limpiar Campos
- Cerrar sesión
- Persistencia: las notas se almacenan en una carpeta aparte y el usuario puede recuperarlas
- El botón de eliminar no funciona si no hay nota seleccionada
